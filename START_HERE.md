## ⚡ WHAT YOU NEED TO DO (3 STEPS)

### STEP 1️⃣ - ADD API KEYS (5 minutes)
**File:** `d:\rapid-response-system\backend\.env`

Replace these 4 lines:
```env
GEMINI_API_KEY="YOUR_GOOGLE_API_KEY"          ← from aistudio.google.com
TWILIO_ACCOUNT_SID="YOUR_TWILIO_SID"          ← from twilio.com/console
TWILIO_AUTH_TOKEN="YOUR_TWILIO_TOKEN"         ← from twilio.com/console
TWILIO_PHONE_NUMBER="+14155552671"            ← from twilio.com/console
```

### STEP 2️⃣ - START BACKEND (1 minute)
```powershell
cd d:\rapid-response-system\backend
node server.js
```

**Wait for:** `✅ Firebase Admin SDK initialized successfully`

### STEP 3️⃣ - RUN APP (5 minutes)
```powershell
cd d:\rapid-response-system\mobile\mobile_app
flutter run
```

**App launches automatically!** 🚀
